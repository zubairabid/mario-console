# Mario on Terminal

**Author:** Zubair Abid

### What
A Mario-like console game implemented in Python. Without curses. Ideally, the only dependencies are numpy and colorama.

Done for SSAD Assignment.

### How to run
```
pip install -r requirements.txt
python main.py
```

### Controls
`w, a, d` for jump, left, and right
`q` to quit

### Dependencies
Package    Version
---------- -------
colorama   0.3.9  
numpy      1.15.0
